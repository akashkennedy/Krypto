## Run Locally

Clone the project

```bash
  git clone https://github.com/akashkennedy/Krypto.git
```

Go to the project directory

```bash
  cd Krypto
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
